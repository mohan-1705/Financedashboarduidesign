# 💰 Finance Dashboard UI

A modern and responsive **Finance Dashboard UI** built to visualize financial data, track transactions, and provide insights for better decision-making.

🔗 **Live Demo:**
https://financedashboarduidesign-7vxebjtab-mohans-projects-2e9d73ba.vercel.app/

---

## 🚀 Features

* 📊 Interactive financial dashboard
* 💳 Add & manage transactions
* 📈 Visual charts for balance & spending
* 🧠 Insights panel for better understanding of finances
* 🌗 Dark/Light mode toggle
* 👤 Role-based UI (user selection)
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **UI Components:** ShadCN UI
* **Charts & Visualization:** Custom chart components

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/mohan-1705/Financedashboarduidesign.git
cd Financedashboarduidesign
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
